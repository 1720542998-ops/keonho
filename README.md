  <h3>咏柳</h3>  
  <p>
    [唐]贺知章<br>
    碧玉<sup>[1]</sup>妆<sup>[2]</sup>成一树<sup>[3]</sup>高，<br>
    万条垂下绿丝绦<sup>[4]</sup>。<br>
    <INS>不知细叶谁裁出，</INS><br>
    <INS>二月春风似剪刀。</INS><br>
    <hr>
    <ul>
        <li>诗词注释</li>
    </ul>
    <dl>
        <dt>[1]碧玉：碧绿色的玉。这里用以比喻春天嫩绿的柳叶。</dt>
        <dt>[2]妆：装饰，打扮。</dt>
        <dt>[3]一树：满树</dt>
        <dt>[4]绦（tāo）：用丝编成的绳带。这里指像丝带一样的柳条。</dt>
    </dl>
<ul>
    <li><I>划线诗句的译文</I></li>
</ul>
<p><I>这细细的嫩叶是谁的巧手裁剪出来的呢？原来是那二月里温暖的春风，它就像一把灵巧的剪刀。</I></p>
